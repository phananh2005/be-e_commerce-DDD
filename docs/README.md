# Documentation Index

Tài liệu dùng hiện tại cho dự án e-commerce backend.

## Tài liệu đang dùng
- `README.md` — trang chính của dự án, cách chạy, build nhanh
- `BOUNDED_CONTEXT_STRUCTURE.md` — mô tả bounded context và phân lớp
- `ARCHITECTURE_DIAGRAM.md` — sơ đồ package/layer theo cấu trúc hiện tại
- `CLOUDINARY_IMPLEMENTATION.md` — hướng dẫn tích hợp upload ảnh
- `HELP.md` — mẹo hỗ trợ nhanh / ghi chú vận hành
- `FRONTEND_WORK.md` — mô tả phạm vi và cấu trúc công việc frontend
- `FRONTEND_AI_PROMPT.md` — bản brief/prompt copy-paste để AI bắt đầu code frontend
- `FRONTEND_BACKEND_REFERENCE.md` — base URL, endpoint thực tế, response JSON, auth/token
- `HELP.md` — mẹo hỗ trợ nhanh và vận hành

## Tài liệu đã lưu trữ
- `docs/archived/` — các báo cáo session, quick cards, bản migration cũ và tài liệu nháp

## Gợi ý đọc nhanh
1. Đọc `README.md` để biết cách chạy dự án
2. Đọc `BOUNDED_CONTEXT_STRUCTURE.md` để hiểu kiến trúc nghiệp vụ
3. Đọc `ARCHITECTURE_DIAGRAM.md` nếu cần map package tới chức năng
4. Đọc `CLOUDINARY_IMPLEMENTATION.md` khi làm việc với upload ảnh
5. Đọc `FRONTEND_WORK.md` khi cần mô tả scope frontend / màn hình / API

## Ghi chú dọn dẹp
Các file cũ trong `docs/archived/` là tài liệu lịch sử, không cần cập nhật cho luồng phát triển hiện tại.
